# 🌍 Web Scraping Project

This project demonstrates web scraping using Python on both static and dynamic websites.

---

## 📌 1. Static Web Scraping (BeautifulSoup)

Scraped country data from:
https://www.scrapethissite.com/pages/simple/

### Data Extracted
- Name
- Capital
- Population
- Area

### Output
Data is saved in `countries.csv`

---

## 📌 2. Dynamic Web Scraping (Selenium)

Scraped quotes from:
https://quotes.toscrape.com/js/

### Data Extracted
- Quote
- Author

### Features
- Implemented pagination (Page 1 to Page 10)
- Handled JavaScript-rendered content

### Output
Data is saved in `quotes_all_pages.csv`

---

## 🛠️ Tools Used
- Python
- Requests
- BeautifulSoup
- Selenium
